# BLOCKCHAIN
# Простая реализация блокчейна на Go

Базовый пример реализации блокчейна на языке Go с использованием криптографического хеширования SHA-256.

## 🚀 Быстрый старт

### Запуск

```bash
go run main.go
```

## 🧱 Архитектура

### Block (Блок)

Структура, представляющая отдельный блок в цепочке:

```go
type Block struct {
    Timestamp     int64   // Время создания блока
    Data          []byte  // Данные блока
    PrevBlockHash []byte  // Хеш предыдущего блока
    Hash          []byte  // Хеш текущего блока
}
```

### Blockchain (Блокчейн)

Структура, содержащая цепочку блоков:

```go
type Blockchain struct {
    blocks []*Block  // Список блоков
}
```

## 💡 Особенности

- ✅ Использование SHA-256 для хеширования
- ✅ Временные метки для каждого блока
- ✅ Связь между блоками через хеши
- ✅ Невозможность изменения данных без пересчета хешей
- ✅ Простая и понятная структура

## 📚 Основные концепции

### Хеширование
Каждый блок содержит хеш, вычисленный на основе его данных и хеша предыдущего блока. Это обеспечивает неизменность цепочки.

### Генезис-блок
Первый блок в цепочке, не имеющий предыдущего блока (PrevBlockHash = []byte{}).

### Цепочка блоков
Каждый новый блок ссылается на хеш предыдущего, создавая непрерывную последовательность.
